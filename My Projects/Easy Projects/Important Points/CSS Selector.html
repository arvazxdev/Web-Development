Important Points About CSS Selectors

  • ".Class" is uses for Class Selector
  • "#ID" is uses before ID Selector
  • "'Element' > 'Element'" is uses for Child Selector
    • It execute with direct child
  • "'Element' 'Element'" is uses for Descendant Selector
  • "*" is uses for Universal Selector
  • "'Element':'Any Pseudo Class'" is uses for Pseudo Selector
