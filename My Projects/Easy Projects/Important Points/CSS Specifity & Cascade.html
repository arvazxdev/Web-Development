Important Points About CSS Specifity & Cascade

  1. The Cascade Algorithm: The Cascade is an algorithm to solve conflicts where multiple CSS rules apply to an HTML element.

  2. Specifity Calculation: An algorithm that determines the strongest and weakest match.

  3. Effect of Position: The order in which CSS rule appears.

  4. Origin: The styles that are defaultally applied by browser.

The Cascade Algorithm

  • If  '!Important' keyword added in any property (before semicolon';' of the property) , 10,000 specifity added to the selector
  • The overwritted property written in CSS programme is apply based on highest specifity

Specifity Calculation

  • The Specifity of universal selector is '0'
  • The Specifity of element selector and pseudo-element are '1'
  • The Specifity of class selector, attribute selector and psedo class are '10'
  • The Specifity of ID selector is '100'
  • The Specifity of inline style is '1000'

Effect of Position

  • If the specifity of the many selectors is equal, the property appear which is written at last and last in the CSS programme

Origin

  • Some of styles automatically applied by browser
